Definition of Counting

Counting is a subset of combinatorics that usually involves counting the possibility of certain events. It is important because it provides insights into the number of possible outcomes with respect to the number of decision. An efficient way of counting is necessary to handle large masses of statistical data and for an understanding of probability.


Basic Rules of Counting

Addition and multiplication rules are used to calculate the possible outcomes of one single event(choosing one fruit once). When choosing multiple fruits or arrange the order of multiple fruits, we need to use combination or permutation, because for each individual selection or arrangement, the size of the original sample changes.
